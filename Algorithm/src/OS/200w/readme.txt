本目录为200万一体使用的uboot、内核、FPGA文件，说明如下：

uboot.img：Uboot升级包，其中已经包含了主和从的uboot
uImage.master：一体机主的内核
uImage.slave：一体机从的内核
fpga.xsvf：FPGA上运行的程序文件
